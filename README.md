## 🚀 Installation & Utilisation

### Prérequis
- Node.js 18+ 
- npm ou pnpm
- Git

### Installation

```bash
# Cloner le repository
git clone https://github.com/rayhan-maouaci/portfolio.git
cd portfolio

# Installer les dépendances
pnpm install
# ou
npm install
```

### Développement

```bash
# Démarrer le serveur de développement
pnpm dev
# ou
npm run dev

# Le portfolio sera disponible à http://localhost:3000
```

### Build Production

```bash
# Construire pour la production
pnpm build
# ou
npm run build

# Prévisualiser la build
pnpm preview
# ou
npm run preview
```
